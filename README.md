# mise-tasks

This directory contains custom tasks for [mise](https://mise.jdx.dev/), designed to automate project setup and configuration.

## Tasks

### [`init:python`](./init/python)
Initializes a new Python project or migrates an existing one using `uv` and `mise`.

**Usage:**
```bash
mise init:python [-p --python <version>] [dir]
```
